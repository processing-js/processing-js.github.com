Class PVector
Name

normalize()
Examples 
PVector v = new PVector(10, 20, 2);
v.normalize();
println(v);  // Prints "[ 0.4454354, 0.8908708, 0.089087084 ]"
Description

Normalize the vector to length 1 (make it a unit vector).
Syntax 
vec.normalize()
Parameters
vec any variable of type PVector
Returns None
