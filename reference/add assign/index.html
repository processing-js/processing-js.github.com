Name

+= (add assign)
Examples 
int a = 50;

int b = 23;

a += b; // sets "a" to 73
Description Combines addition with assignment. The expression a += b is equivalent to a = a + b.
Syntax 
value1 += value2
Parameters
value1 int or float
value2 any numberical value the same datatype as value1
Usage Web & Application
Related = (assign)
+ (add)
-= (subtract assign)
