Name

-= (subtract assign)
Examples 
int a = 50;

int b = 23;

a -= b; // sets "a" to 27
Description Combines subtraction with assignment. The expression a -= b is equivalent to a = a - b.
Syntax 
value1 -= value2
Parameters
value1 int or float
value2 int or float
Usage Web & Application
