--- layout: withcc title: toUpperCase() | reference ---
Class String
Name

toUpperCase()
Examples 
String str1 = "Cccp"; 
String str2 = "Rabbit"; 
str1 = str1.toUpperCase();
str2 = str2.toUpperCase();
println(str1 + ":" + str2);  // Prints CCCP:RABBIT
Description Converts all of the characters in the string to uppercase. For example, "abc" will convert to "ABC".
Syntax 
str.toUpperCase()
Parameters
str String: any variable of type String
Returns String
