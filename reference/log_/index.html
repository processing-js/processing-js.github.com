--- layout: withcc title: log() | reference ---
Name

log()
Examples 
void setup() {
  int i = 12;
  println(log(i));
  println(log10(i));
}

// Calculates the base-10 logarithm of a number
float log10 (int x) {
  return (log(x) / log(10));
}
Description

Calculates the natural logarithm (the base-e logarithm) of a number. This function expects the values greater than 0.0.
Syntax 
log(value)
Parameters
value int or float: number must be greater then 0.0
Returns float
