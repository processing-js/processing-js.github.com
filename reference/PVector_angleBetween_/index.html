Class PVector
Name

angleBetween()
Examples 
PVector v1 = new PVector(10, 20);
PVector v2 = new PVector(60, 80); 
float a = PVector.angleBetween(v1, v2);
println(degrees(a));  // Prints "10.304827"
Description

Calculates and returns the angle (in radians) between two vectors.
Syntax 
PVector.angleBetween(v1,v2)
Parameters
vec any variable of type PVector
v1 PVector: the x, y, and z components from a PVector object
v2 PVector: the x, y, and z components from a PVector object
Returns float
