Class PVector
Name

z
Examples 
PVector v;

void setup() {
  size(100, 100);
  v = new PVector(20.0, 30.0, 40.0);
  println(v.x);  // Prints "20.0"
  println(v.y);  // Prints "30.0"
  println(v.z);  // Prints "40.0"
  v.z = 50.0; 
  println(v.z);  // Prints "50.0"
}
Description

The z component of the vector. This field (variable) can be used to both get and set the value (see above example.)
Syntax 
vec.z
Parameters
vec any variable of type PVector
