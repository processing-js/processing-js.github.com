Name

>= (greater than or equal to)
Examples 
int a = 23;

int b = 23;

if(a >= b) {

  println("variable a is greater or equal to variable b ")

}
Description Tests if the value on the left is larger than the value on the right or if the values are equivalent.
Syntax 
value1 >= value2
Parameters
value1 int or float
value2 int or float
