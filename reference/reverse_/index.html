--- layout: withcc title: reverse() | reference ---
Name

reverse()
Examples 
String sa[] = { "OH ", "NY ", "MA ", "CA "};
sa = reverse(sa);
print(sa);  // Prints "CA MA NY OH "
Description

Reverses the order of an array.
Syntax 
reverse(array)
Parameters
array boolean[], byte[], char[], int[], float[], or String[]
Returns Array (the same datatype as the input)
