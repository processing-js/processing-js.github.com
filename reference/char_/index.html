Name

char()
Examples 
int i = 65;

char c = char(i);

println(i + " : " + c);  // Prints "65 : A"



byte b = 65;

char c = char(b);

println(b + " : " + c);  // Prints "65 : 65"
Description Converts a primitive datatype or array to a numeric character representation.
Syntax 
char(val)
Parameters
val int, byte, int[], byte[]
Returns char
